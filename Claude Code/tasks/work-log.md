# 工作日志

> **每次对话必须更新本文件**。记录本次会话做了什么、改了什么、学到什么。
> 作用：让下一轮会话能快速接上，不用重新猜上下文。
> 格式：按时间倒序，最新一条在最上面。

---

## [YYYY-MM-DD HH:MM] 一句话标题

**做了什么**：
- 动作 1
- 动作 2

**改了哪些文件**：
- `path/to/file` — 改了什么

**学到什么**：
- 本次发现的规律 / 踩坑 / 纠正

**下次注意**：
- 下次应该怎么做 / 避免什么

## 维护原则

- **每次对话至少写一条**，哪怕只是"今天没改动，只是确认了一下状态"
- 每条必须有：做了什么 / 改了哪些文件 / 学到什么 / 下次注意
- 最新一条在最上面（时间倒序）
- 超过 30 天的旧条目合并归档到 `progress.md`
- 不写流水账（"先读了文件 A，再读了文件 B"），只写结论和收获
